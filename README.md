# Описание

Данная работа является курсовым проектом. В качестве учебной практики было разработана игровая платформа-аркада с выбором из трёх игр: Змейка/Тетрис/Пинг-Понг.

## Технологии
С++ - Основная логика и алгоритмы<br>
CMake - сборщик проекта и библиотек<br>
RayLib - графическая библиотека для отрисовки игровых объектов и интерфейса.
